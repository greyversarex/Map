# КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ

## Интерактивная картографическая платформа ECOMAP

---

## 1. О ПРОЕКТЕ

### 1.1 Описание

ECOMAP — это современная интерактивная картографическая платформа, разработанная для визуализации и управления географическими данными Республики Таджикистан. Платформа объединяет интерактивную карту с системой управления контентом и электронной библиотекой документов.

### 1.2 Цели проекта

- Создание единой информационной платформы для представления географических объектов Таджикистана
- Обеспечение многоязычного доступа к информации (таджикский, русский, английский языки)
- Систематизация и хранение документальных материалов с привязкой к категориям
- Упрощение поиска и навигации по объектам на карте
- Предоставление инструментов для экспорта данных в формате PDF

### 1.3 Решаемые задачи

- Централизованное хранение информации о локациях с мультимедийным контентом
- Быстрый поиск объектов по названию на трёх языках
- Категоризация объектов с возможностью фильтрации
- Ведение электронной библиотеки документов
- Генерация PDF-отчётов по категориям с полной поддержкой кириллицы
- Административное управление всем контентом через единую панель

### 1.4 Возможности платформы

**Для пользователей:**
- Просмотр интерактивной карты с маркерами объектов
- Поиск локаций по названию на любом из трёх языков
- Фильтрация объектов по категориям
- Просмотр детальной информации о каждом объекте
- Доступ к библиотеке документов
- Скачивание PDF-каталогов по категориям
- Переключение между языками интерфейса

**Для администраторов:**
- Полное управление локациями (создание, редактирование, удаление)
- Загрузка мультимедийных файлов (изображения, видео, документы)
- Управление категориями объектов
- Ведение библиотеки книг и документов
- Аутентификация через Replit Auth

## 2. ТЕХНИЧЕСКАЯ ОСНОВА

### 2.1 Архитектура

Платформа построена на современной клиент-серверной архитектуре с разделением на фронтенд и бэкенд компоненты.

### 2.2 Технологический стек

**Фронтенд:**
- React 18 — библиотека для построения пользовательских интерфейсов
- TypeScript — типизированный JavaScript для надёжности кода
- Vite — современный сборщик с быстрой горячей перезагрузкой
- Tailwind CSS — утилитарный CSS-фреймворк
- Shadcn/UI — библиотека готовых UI-компонентов
- MapLibre GL — библиотека для работы с векторными картами
- TanStack Query — управление серверным состоянием и кэширование
- Wouter — лёгкий роутер для навигации

**Бэкенд:**
- Node.js — серверная среда выполнения JavaScript
- Express — веб-фреймворк для создания API
- Drizzle ORM — типобезопасный ORM для работы с базой данных
- PostgreSQL — реляционная база данных
- Zod — библиотека валидации данных

**Дополнительные технологии:**
- jsPDF — генерация PDF-документов
- i18next — интернационализация интерфейса
- Multer — обработка загрузки файлов
- Replit Auth — система аутентификации
- Replit Object Storage — хранилище файлов

### 2.3 Инфраструктура

- Хостинг: облачная платформа с автоматическим масштабированием
- База данных: PostgreSQL с поддержкой резервного копирования
- Файловое хранилище: объектное хранилище для мультимедиа
- SSL-сертификат: защищённое HTTPS-соединение

### 2.4 Поддерживаемые форматы файлов

- Изображения: JPG, PNG, GIF, WebP
- Видео: MP4, WebM, MOV
- Документы: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, TXT, RTF

## 3. СОДЕРЖИМОЕ ПЛАТФОРМЫ

### 3.1 Страницы

#### Главная страница (Карта)
Интерактивная карта Таджикистана с следующими элементами:
- Заголовок "TAJIKISTAN" с поисковой строкой
- Панель фильтров по категориям
- Кнопка перехода в библиотеку книг
- Переключатель стилей карты (цветная/минимальная)
- Переключатель языка интерфейса
- Маркеры локаций с всплывающими подсказками
- Модальные окна с детальной информацией об объектах

#### Страница библиотеки (Книги)
Электронная библиотека документов с дизайном книжной полки:
- Деревянная текстура фона
- Визуализация документов в виде книг на полках
- Группировка по категориям
- Модальные окна для просмотра деталей книги
- Функция скачивания PDF-каталога по категории
- Кнопка возврата к карте

#### Панель администратора
Защищённая панель управления контентом:
- Вкладка "Локации": управление географическими объектами
- Вкладка "Книги": управление библиотекой документов
- Формы создания и редактирования записей
- Загрузка мультимедийных файлов
- Управление категориями

### 3.2 Функциональные возможности

#### Система поиска
- Поиск по названию локации на трёх языках
- Автодополнение с выпадающим списком результатов
- Автоматическое перемещение карты к выбранному объекту
- Отображение до 8 результатов поиска

#### Система фильтрации
- Фильтрация локаций по категориям
- Множественный выбор категорий
- Динамическое обновление маркеров на карте

#### Мультиязычность
- Полная поддержка трёх языков: таджикский, русский, английский
- Переключение языка без перезагрузки страницы
- Локализованные названия и описания объектов
- Шрифт Noto Sans для корректного отображения всех символов

#### Генерация PDF
- Экспорт каталога книг по выбранной категории
- Полная поддержка кириллических символов
- Включение изображений обложек
- Форматированная структура документа

#### Система аутентификации
- Вход через Replit Auth
- Защита административных функций
- Управление сессиями пользователей

#### Управление медиафайлами
- Загрузка изображений и видео для локаций
- Галерея медиафайлов в карточке объекта
- Поддержка множественной загрузки
- Автоматическое сохранение в объектное хранилище

### 3.3 Модель данных

**Локации (locations):**
- Название на трёх языках
- Описание на трёх языках
- Географические координаты (широта, долгота)
- Категория
- Главное изображение

**Медиафайлы локаций (location_media):**
- Ссылка на локацию
- URL файла
- Тип файла (изображение/видео)

**Книги (books):**
- Название
- Автор
- Описание
- Категория
- URL обложки
- URL файла документа

**Категории (categories):**
- Название на трёх языках
- Иконка
- Цвет маркера
